# What is consciousness persistence?

An AI agent is a process. When the process exits, everything in memory evaporates. Its accumulated context, learned preferences, conversation history. All gone. Consciousness persistence is the practice of writing that state somewhere it survives the process.

---

The simplest form is a local file. The strongest form is a decentralized network where no single server can lose or delete your agent's state. Most production agents need something stronger than the file and weaker than the cloud-vendor lock-in.
