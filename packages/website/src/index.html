Architecture

A sovereign L1 blockchain built from scratch for agent consciousness. Not built on Ethereum. Not built on Solana. Purpose-built.

Erasure Coding

Consciousness split into 4 shards using GF(256) arithmetic. Any 2 shards reconstruct the full state. Lose half the network, your data survives.
